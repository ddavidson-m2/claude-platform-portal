# Claude Platform Portal

Enterprise Management System UI - Platform Portal

## Features

- **Dashboard** - Overview with quick stats and navigation cards
- **Platform Manager** - 3-column drill-down for Applications, Projects, Services
- **Organization Manager** - Manage organizations and teams
- **Event Notifications** - System alerts and notifications
- **Jim Memorial** - Tribute to the founder

## Quick Start
```bash
# Clone the repo
git clone https://github.com/ddavidson-m2/claude-platform-portal.git
cd claude-platform-portal

# Install dependencies
npm install

# Run development server
npm run dev
```

Opens at http://localhost:5173

## Tech Stack

- React 18
- Vite
- Tailwind CSS
- Lucide React Icons
